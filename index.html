    • ¿Cómo puedo utilizar esta app?

    Hay muchas formas de utilizar los GENERATOR QR:

    Generar un Qr con un enlace para compartir.
    Conocer el contenido de un estante o archivero.
    Funcionalidad de un artefacto o electrodoméntico.
    Indicar la ubicación de un objeto o lugar.
    Mostrar el valor o precio de un producto.
    Dar a conocer todo tipo de información para que este
    disponible para quien la necesite; clientes, visitas, huespedes.
    Ya sea; en una casa, un hotel, una oficina o un local.

    A quien esta dirigida esta herrmienta:

      Locales de todo tipo/ Oficinas/ Empresas/ Hoteles/ Casas

    Existen infinidad de alternativas en las que puedes usar
    este tipo de códigos, solo necesitas generarlos con el texto
    que quieras comunicar, imprimirlo y colocarlo donde las personas
    puedan escanearlo con su celular y acceder a dicha información.
